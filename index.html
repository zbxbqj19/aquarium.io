아두이노로 스마트 아쿠아리움을 만들어보자

스마트하게 관리되는 수족관, 이제 나만의 아쿠아리움을 만들어보세요.

온도 센서 (DS18B20)

DS18B20 → Arduino UNO 연결: VCC(빨강) = 5V, GND(검정) = GND, DATA(노랑) = D3

pH 센서

pH 센서 모듈 → Arduino UNO 연결: VCC = 5V, GND = GND, Signal = A0

산소 센서 (DO Sensor)

산소 센서 → Arduino UNO 연결: VCC = 5V, GND = GND, Signal = A1

조도 센서 (LDR)

LDR 모듈 → Arduino UNO 연결: VCC = 5V, GND = GND, Signal = A2

히터 (릴레이 제어)

릴레이 모듈 → Arduino UNO 연결: VCC = 5V, GND = GND, IN = D5 → 히터 제어

산소기 (릴레이 제어)

릴레이 모듈 → Arduino UNO 연결: VCC = 5V, GND = GND, IN = D7 → 산소기 제어

LED 조명(네온 픽셀)

LED 모듈 → Arduino UNO 연결: VCC = 5V, GND = GND, Signal = D9

알람 (부저)

부저 → Arduino UNO 연결: VCC = 5V, GND = GND, Signal = D11
